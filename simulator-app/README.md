# Warehouse Simulator App

## Run locally
1. `cd simulator-app`
2. `npm install`
3. `npm run dev`
4. Open the printed local URL (typically `http://localhost:5173/warehouse-simulator/`).

## Build for production
1. `cd simulator-app`
2. `npm install`
3. `npm run build`

The build output is written to `../warehouse-simulator` so Cloudflare Pages serves it directly at `/warehouse-simulator/`.
